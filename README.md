# penicillin-bioreactor-optimization
We use a real industrial-style penicillin fermentation dataset (time, RPM, aeration, feed rate, OUR, CER, biomass, product concentration, etc.) to build a predictive and optimization system.  Predicts biomass and penicillin concentration over time.   Recommends optimal operating conditions (RPM, aeration, feed rate) to maximize final yield.
